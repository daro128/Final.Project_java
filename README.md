# Final.Project_java
Java : Stock System 

Intro  

The system can check business stock and profit, and update stock. It can check current stock and profit by showing item’s ID, name, quantity, buy price, sell price, profit per item, and total profit for each item. 

It also has an alert flag for items that are out of stock or almost out. Moreover, it has functionality that can update the stock such as adding item quantity, adding new items, and updating item quantity in stock. 

Furthermore, the  system Supports two main user role: Admin and staff, and requires users to log in using a username and password to access the system. Admin has full control over stock and system management, while Staff is responsible for daily stock and sales operation . 

The last function is monthly profit. It shows the data on profits and sales of items in each month.
